### [Sushi Lang](https://github.com/bigwhale/sushi-lang)

A compiled programming language with a Lark parser frontend and LLVM backend that produces native binaries. Built as a learning exercise in compiler design, combining safety features inspired by a blend of C, Go, Rust, and Python.

**Features:**
- Static typing with compile-time borrow checking
- Type-safe error handling with Result types
- Generics with compile-time monomorphization
- Pattern matching on enums
- Native code generation via LLVM
